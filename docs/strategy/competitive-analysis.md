# Análisis competitivo — Coco

_Última actualización: 2026-04-17_

---

## Mapa del mercado

### Categoría A — Datos técnicos para expertos
Apps con mucha información, pensadas para usuarios avanzados. Coco no compite aquí.

| App | Fortaleza | Debilidad para el usuario medio |
|-----|-----------|--------------------------------|
| **Windy** | Datos en tiempo real, visualizaciones, cobertura global | Muy complejo, requiere saber interpretarlo |
| **Windguru** | Tablas detalladas, histórico, fiable | Interfaz anticuada, curva de aprendizaje alta |
| **PredictWind** | Muy preciso, usado por navegación | Caro, demasiado técnico |
| **XCWeather** | Técnico, buena previsión | No orientado a deportes acuáticos |

### Categoría B — Apps de surf/paddle con datos
Apps orientadas a deportes acuáticos pero que siguen siendo técnicas.

| App | Fortaleza | Debilidad |
|-----|-----------|-----------|
| **Surfline** | Previsiones de surf muy específicas, cámaras de playa | Solo surf, caro (premium), no disponible en muchas zonas europeas |
| **Magic Seaweed / MSW** | Comunidad, previsiones de surf | Solo surf, interfaz densa |
| **Swellnet** | Australia/NZ, comunidad | Limitado geográficamente |
| **iWindsurf / iKite** | Orientado a windsurf/kite | Nicho muy específico |

### Categoría C — El hueco que ocupa Coco
Apps que explican las condiciones en lenguaje accesible, para el usuario no experto.

**Prácticamente vacío en España y Europa para paddle surf.**

---

## Conclusión estratégica

Coco no necesita ser mejor que Windy en datos.
Necesita ser mejor que cualquier app en claridad y accesibilidad para el usuario medio.

El diferencial no es técnico — es editorial y de experiencia.
